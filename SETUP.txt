------------------------------

4. Klik tab "Import"
5. Pilih file kantor.sql dari folder Database-PHP
6. Klik tombol "Go" atau "Import"

ATAU gunakan command line:

USE kantor;
source kantor.sql;

RUMUS PERHITUNGAN GAJI
======================
Total Gaji = Gaji Pokok + Tunjangan + Total Lembur + Bonus

Dimana:
- Gaji Pokok: dari Jabatan
- Tunjangan: dari Jabatan
- Total Lembur: Jam Lembur Ã— Tarif Lembur
- Bonus: Gaji Pokok Ã— Presentase Rating

Contoh:
- Karyawan: Budi (Manajer)
- Gaji Pokok: 10.000.000
- Tunjangan: 3.000.000
- Jam Lembur: 5 jam
- Tarif Lembur: 25.000/jam
- Rating: 5 (20% bonus)

Perhitungan:
- Total Lembur = 5 Ã— 25.000 = 125.000
- Bonus = 10.000.000 Ã— 20% = 2.000.000
- Total Gaji = 10.000.000 + 3.000.000 + 125.000 + 2.000.000 = 15.125.000
